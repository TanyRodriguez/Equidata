# 📌 Descripción del Proyecto

**EquiData Global** es una plataforma inteligente diseñada para analizar la violencia de género a nivel **América Latina y por país**, transformando datos en información accionable para apoyar la toma de decisiones.

El objetivo del proyecto no es únicamente visualizar datos, sino **convertirlos en conocimiento útil**, permitiendo identificar patrones, tendencias y zonas críticas mediante un sistema de análisis interactivo.

Este proyecto integra múltiples fuentes de datos oficiales internacionales y nacionales, los procesa y los presenta en un **dashboard interactivo** con capacidades analíticas.

En resumen, **no estamos creando solo un dashboard**, estamos desarrollando una **herramienta de inteligencia de datos** que ayuda a comprender el problema y facilitar decisiones basadas en evidencia.

---

# 🎯 Objetivo del Proyecto

Desarrollar una plataforma de inteligencia de datos que permita:

- Identificar países o regiones con mayor incidencia
- Detectar patrones por edad, tipo de violencia y tiempo
- Analizar tendencias históricas
- Comparar contextos entre países
- Generar alertas en zonas críticas
- Apoyar decisiones institucionales basadas en datos
